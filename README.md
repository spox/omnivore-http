# omnivore-http

Omnivore HTTP source implementation

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  omnivore-http:
    github: spox/omnivore-http
```


## Usage


```crystal
require "omnivore-http"
```
